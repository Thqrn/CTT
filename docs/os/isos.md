# ISOs

## 0. Backing up data

If you're gonna install an operating system and wipe your on a disk, consider checking these places for stuff you might've left behind:

* .minecraft folder and prism/multimc instances, contains screenshots, logs, packs..
* Web browser history, bookmarks, list of extensions
* Configuration files for program you've extensively personalized

## 2. Flashing the installation media


* [Rufus](https://rufus.ie/en/) - can tweak some Windows settings pre-install
* [balenaEtcher](https://etcher.balena.io) - works better with flashing Linux from [my](/contact.md#couleur) experience
* [Ventoy](https://ventoy.net) - can hold multiple ISOs (literally drag and drop it to the partition), then you choose what ISO to boot on at startup.
