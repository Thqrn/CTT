# mpv

Extensible and keyboard driven media player

## Installation 

* <https://github.com/mpv-player/mpv#mpv>
* <https://mpv.io/manual/stable> invaluable

linux is first class, for windows it has no officially merged file associator or [installer to PATH](https://github.com/mpv-player/mpv/pull/15912) yet, see [this guide](https://kenhv.com/blog/setting-up-mpv)

# Configuration

See [The Wiki's](https://thewiki.moe/tutorials/mpv/#config-overview) documentation
